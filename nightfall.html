Nightfall

This group project focused on creating a cinematic trailer for a survival game set in a dinosaur infested world. The scene is set in a dark, rainy environment. The goal was to immerse the viewer in a sense of danger and chaos. We combined a 3D environment with the 2D characters that were designed to be more stylized or simplified.
